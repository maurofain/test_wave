# ✅ 1. Motion Detection sulla DFR1154

La DFR1154 monta **ESP32‑S3**, quindi puoi eseguire motion detection direttamente **on‑board** tramite:

### **A) Algoritmo software (OpenCV‑lite / frame differencing)**

Sul microcontrollore puoi implementare un motion detection semplice: differenza tra frame consecutivi + threshold.

Oppure puoi usare modelli ML leggeri via **Edge Impulse**, supportato ufficialmente dal modulo.  
→ La wiki conferma la possibilità di usare modelli personalizzati di object detection tramite EdgeImpulse. [\[wiki.dfrobot.com\]](https://wiki.dfrobot.com/dfr1154/docs/21367)

### **B) Sensore PIR esterno**

DFRobot nei forum discute proprio l’uso di un PIR con la DFR1154.  
È la soluzione **più affidabile** per rilevare avvicinamento persone, ma richiede hardware aggiuntivo. [\[dfrobot.com\]](https://www.dfrobot.com/forum/?t=1207)

# ✅ 2. Inviare fotogrammi via USB all’host

Il modulo può funzionare come **USB Webcam (UVC)**:  
DFRobot fornisce un esempio ufficiale: **USBWebCamera**.

Lo conferma il repository *DFR1154\_Examples → USBWebCamera*, dove:

*   Devi flashare un **firmware BIN** USB‑Webcam sulla DFR1154
*   Collegare la scheda via USB al PC
*   Sul computer puoi usare uno script es. **Image\_reception.py** per ricevere i frame (via OpenCV).    [\[github.com\]](https://github.com/DFRobot/DFR1154_Examples/blob/master/5.4%20USBWebCamera/README.md), [\[github.com\]](https://github.com/DFRobot/DFR1154_Examples/tree/master/5.4%20USBWebCamera)

# 🔧 3. Librerie da usare

## **Sul modulo DFR1154 (ESP32‑S3):**

Usi l’SDK **ESP‑IDF** (richiesto dagli esempi ufficiali).  
Gli esempi USB webcam derivano da Espressif:  
→ <https://github.com/espressif/esp-iot-solution/tree/master/examples/usb/device/usb_webcam> [\[github.com\]](https://github.com/DFRobot/DFR1154_Examples/blob/master/5.4%20USBWebCamera/README.md)

All’interno puoi usare:

*   Driver della camera (built‑in ESP32‑S3)
*   Eventuali modelli ML (TinyML / Edge Impulse)
*   Algoritmo di motion detection custom

***

## **Sul PC (Host) via USB:**

Gli esempi richiedono l’uso di:

### **OpenCV + Python**

Come indicato chiaramente nella documentazione:

*   Installare Python
*   Installare `opencv-python`
*   Aprire il file **Image\_reception.py** per ricevere l’immagine via USB    [\[gitee.com\]](https://gitee.com/dfrobot/DFR1154_Examples/tree/master/5.4%20USBWebCamera)

Esempio minimale (host):

```python
import cv2

cap = cv2.VideoCapture(1)   # cambia device_id se necessario

while True:
    ret, frame = cap.read()
    if not ret:
        break
    cv2.imshow("DFR1154 Stream", frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
```
